CH 12: Object-Oriented Programming in JavaScript (OOP)

Object-Oriented Programming a programming style based on the idea encapsulating code into "objects" which are primarily methods and variables that would characterize an object in the real world. For instance, if I wanted to code a user profile, the object could have variables to represent the name, age, and contact info of the user. the contact info variable could be another object that encapsulates the address, phone number, and email address of the user.
