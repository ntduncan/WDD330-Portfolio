My Test Code Example:

let name = 'nate';
let description = 'cool coder kid';

localStorage.setItem(name, description);
bleh.textContent = localStorage.getItem(name);

//Test:
description = "The Cool Coder Kid"

//Result: Doesnâ€™t change the output because localStorage was already set

LocalStorage Methods:
setItem(key, value) - Puts an item in local storage given a key
getItem(key) - Returns the value of an item in local storage given a key
removeItem(key) - Removes an item from local storage given a key
clear() - Clears whole local storage

Master Mobile UX

Main Takeaways

  • Mobile design first. Doing it last is like trying to stuff a marshmallow in a piggy bank; Messy and difficult
  • For reachability, main functionality should be lower on a mobile UI
  • Try to minimize the need for user interaction
  • upscale elements that might be hard to work with using thumbs
  • DO NOT USED POPUPS OR FULL SIZED ADDS
