The Window Object

The window object is the browser environments global object. It represents the state of the browser window. Through the Window object, one can access information and control certain elements of the browser in javaScript. Because the window object is global, you can access it anywhere in you frontend code.


Here are 3 examples of window methods:

  • prompt() - Accepts an input from the user through a pop up window
  • alert() - Sends a pop up message to the user
  • confirm() - Gives the user a binary repsonse option through a pop up window
